# PocketList 🛒

PocketList es una aplicación móvil sencilla y eficiente diseñada para ayudarte a organizar tus listas de compras. Ya sea para el supermercado, la ferretería o preparativos para ocasiones especiales, PocketList te permite llevar el control total de lo que necesitas adquirir para que nunca olvides nada.

## ✨ Características Principales

- **Gestión de Listas:** Crea múltiples listas de compras personalizadas.
- **Presupuesto:** Realiza un seguimiento de tus gastos asignando presupuestos y precios a los artículos.
- **Exportación a PDF:** Genera documentos PDF de tus listas para imprimir o compartir de forma profesional.
- **Importación/Exportación de Datos:** Comparte tus listas con otros usuarios de la app o realiza copias de seguridad de tu información.
- **Personalización:** Soporte para temas (colores), modo oscuro y multi-idioma (Español e Inglés).
- **Interfaz Moderna:** Diseño limpio basado en Material Design para una experiencia de usuario fluida.
- **Actualizaciones:** Notificación automática cuando haya una nueva versión disponible en la tienda.

## 🚀 Tecnologías Utilizadas

- **Flutter:** Framework para el desarrollo multiplataforma.
- **Dart:** Lenguaje de programación principal.
- **Sqflite:** Base de datos local para almacenamiento persistente.
- **Shared Preferences:** Almacenamiento de configuraciones del usuario.
- **i18n:** Sistema de localización para soporte multi-idioma.
- **Upgrader:** Detección automática de actualizaciones en tienda.

## 🛠️ Instalación y Configuración

1. **Requisitos previos:**
   - Tener instalado [Flutter](https://docs.flutter.dev/get-started/install).
   - Un editor de código como VS Code o Android Studio con los plugins de Flutter/Dart.

2. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/tu-usuario/pocketlist.git
   cd pocketlist
   ```

3. **Instalar dependencias:**
   ```bash
   flutter pub get
   ```

4. **Ejecutar la aplicación:**
   ```bash
   flutter run
   ```
